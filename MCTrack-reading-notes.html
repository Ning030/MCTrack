📌 项目一句话介绍

MCTrack 是一个面向自动驾驶场景的统一 3D 多目标跟踪（MOT）框架， 通过标准化的 BaseVersion 数据格式和可配置的跟踪管线（扩展卡尔曼滤波 + 多策略数据关联）， 在 KITTI、nuScenes、Waymo 三大主流数据集上取得了 SOTA 表现（KITTI 第 1 名、nuScenes 第 1 名、Waymo 第 2 名）， 并引入了创新的运动指标（Motion Metrics）来评估速度和加速度估计质量。该论文被 IROS 2025 收录。

def transform_3dbox2corners(self, global_xyz_lwh_yaw) -> np.ndarray:
    x, y, z, l, w, h, rot = global_xyz_lwh_yaw
    orientation = Quaternion(axis=[0, 0, 1], radians=rot)
    # 构建 8 个角点的局部坐标
    x_corners = np.array([dx1,dx1,dx1,dx1,dx2,dx2,dx2,dx2]) * [1,1,1,1,-1,-1,-1,-1]
    # ... 旋转 + 平移 ...
    corners = np.dot(orientation.rotation_matrix, corners)
    corners[0,:] += x; corners[1,:] += y; corners[2,:] += z
    return corners.T  # shape: (8, 3)

Cost cal_rotation_gdiou_inbev — 旋转广义距离 IoU (RO-GDIoU)

📁 tracker/cost_function.py:34

输入 / 输出

方向名称类型说明
输入box_trkTrajectory跟踪轨迹
输入box_detBBox检测框
输入cal_flagstr"Predict" 或 "BackPredict"
输出ro_gdioufloat旋转 GDIoU 分数（越高越匹配）

实现逻辑

  1. 根据 cal_flag 选择使用预测位置 (_predict) 或融合位置 (_fusion)
  2. 计算两个框的 BEV 角点，使用 cv2.rotatedRectangleIntersection 计算旋转矩形交集面积
  3. 计算最小外接矩形面积 (cv2.minAreaRect) 作为 universe_area
  4. 计算三个组分：IoU、额外面积惩罚（w1）、中心距离惩罚（w2）
  5. 公式：ro_gdiou = iou - w1 * extra_area/universe_area - w2 * center_dist²/diagonal²

关键代码片段

ro_gdiou = (
    iou
    - w1 * extra_area / universe_area
    - w2 * box_center_distance / union_distance
)

算法 / 模式

GDIoU 扩展 — 在标准 GIoU 基础上加入旋转支持和可配置权重
OpenCV — 使用 cv2.rotatedRectangleIntersection 处理旋转矩形交集

实现逻辑

  2. LAPJV 内部使用 Jonker-Volgenant 算法（匈牙利算法的高效实现）
  3. extend_cost=True 允许部分匹配（非强制全匹配）

Greedy 算法

  1. 对每个检测，找到代价最小的轨迹
  2. 若最小代价 ≤ 阈值，建立匹配并将该轨迹列屏蔽
  3. 简单但不保证全局最优

算法 / 模式

线性分配问题 (LAP) — 最优匹配，O(n³) 复杂度
贪心匹配 — 次优但 O(n²) 复杂度

EKF_CV 运动模型

EKF_CA 运动模型

def update(self, z):
    # Predict
    self.x = self.f(self.x)
    self.F = self.getF(self.x)
    self.P = self.F * self.P * self.F.T + self.Q
    # Update (Kalman Gain)
    G = self.P * self.H.T * np.linalg.inv(self.H * self.P * self.H.T + self.R)
    self.x += G * (np.array(z) - self.h(self.x).T).T
    self.P = (self.I - np.matmul(G, self.H)) * self.P
    return np.array(self.x.reshape(self.n))
